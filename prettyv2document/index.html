  How to install

Extra the source .zip file into your htdocs folder (Apache root folder), you can create sub-folder under it and point to the right path.

Open browser and type http://your-host-name/install-path
(Depend on your destination path it should look like http://localhost/prettys3filemanager)

 

1. First installation screen will show

 

 

At this step please read the license agreement, check the agree checkbox and click next!

2. System Requirement

 

 

To install script make sure your server is running PHP version at least 5.4, otherwise you have to upgrade your server first.
Make sure the data folder is writable.
After all requirements are passed, click next and go to step 3.

3. System settings

 

This is most important step and please set up it correctly

 

Ignore the site URL section, that field system will auto detect it for you.

Fill in your App Key and Secret Key and please copy and paste them correctly. For instruction how to find them you click on button Follow Instruction and follow those steps to find out keys.

Choose your region (it must be same with your region when registering S3 service)

After those steps above, click on button Connect to S3 Cloud to check connection status and fetch available buckets, choose one default bucket for your setting.

Choose your login name and set password.
Please keep these information in safe and remember it for next time login. If you forget it, you have to reinstall again.

4. Finish

 

This step is actually a joke :). It's meant all things are ready!

  Manage Bucket/Folder

At the dashboard screen, you can easily to access Buckets at the top dropdown box, also you can create new bucket, a popup for creating bucket will appear, enter name of it then hit create.

 

 

Create bucket popup

 

 

At the content area, you will see all folders will be listed on the left panel, you can access it's file by clicking on it, browse sub-folder or creating a new folder by clicking on create folder at top bucket panel or bottom button.

 

 

Popup create folder will show, choose the target path on the left folder panel, fill the name of it and hit create

 

  Manage Files

All file will be listed under the file management table on the right side, by default application will list the Root / path for files, you can change the folder on the left panel(Click directly on folder name to change the path). You can preview, download or remove them.

 

  Upload Files

Access upload files popup, you click on upload files.

 

 

Popup upload files will appear, choose the destination path on the left panel, choose your files then click start upload

 

  Object Properties

You can access this feature by clicking on file name or under Action/Properties dropdown button.
Here you can modify permissions, view file property detail, change the content type of it.

 

  Change Password

You can change your current login password to a new one, access this feature by clicking on action menus.

 

 

Verify your current password, type and retype it then hit update. By the next login session, lets use your new password.
