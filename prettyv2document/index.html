  How to install

Extra the source .zip file into your htdocs folder (Apache root folder), you can create sub-folder under it and point to the right path.

Open browser and type http://your-host-name/install-path
(Depend on your destination path it should look like http://localhost/prettys3filemanager)

 

1. First installation screen will show

 

 

At this step please read the license agreement, check the agree checkbox and click next!

2. System Requirement

 

 

To install script make sure your server is running PHP version at least 5.4, otherwise you have to upgrade your server first.
Make sure the data folder is writable.
After all requirements are passed, click next and go to step 3.

3. System settings

 

This is most important step and please set up it correctly

 

Ignore the site URL section, that field system will auto detect it for you.

Fill in your App Key and Secret Key and please copy and paste them correctly. For instruction how to find them you click on button Follow Instruction and follow those steps to find out keys.

Choose your region (it must be same with your region when registering S3 service)

After those steps above, click on button Connect to S3 Cloud to check connection status and fetch available buckets, choose one default bucket for your setting.

Choose your login name and set password.
Please keep these information in safe and remember it for next time login. If you forget it, you have to reinstall again.

4. Finish

 

This step is actually a joke :). It's meant all things are ready!
